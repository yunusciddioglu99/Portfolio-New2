Turna B.V. - Vangplan Systeem

Full-stack webapplicatie voor pluimveebedrijf Turna B.V. voor het beheren van weekplanningen, werkbonnen, facturatie, kilometerregistratie, omzetanalyse, werknemers, uren en verlofbeheer. Het systeem automatiseert administratieve processen en biedt real-time inzicht in bedrijfsgegevens.

Facturatiesysteem: Automatisch facturen genereren, PDF met BTW/G-rekening, e-mail via SMTP
Km-rapport: Rittenregistratie met OpenStreetMap API, PDF-export per maand
Omzet Dashboard: Interactief dashboard met omzet per maand/kwartaal/jaar
Verlofregistratie: Verlofaanvragen bijhouden met pro-rata rechten en saldo
TypeScript React Node.js Express Firebase Tailwind CSS Redux Toolkit
